# Installation

## Stable release

To install backmap, run this command in your terminal:

```sh
uv add backmap
```

Or if you prefer to use `pip`:

```sh
pip install backmap
```

## From source

The source files for backmap can be downloaded from the [Github repo](https://github.com/ranjanmannige/backmap).

You can either clone the public repository:

```sh
git clone git://github.com/ranjanmannige/backmap
```

Or download the [tarball](https://github.com/ranjanmannige/backmap/tarball/master):

```sh
curl -OJL https://github.com/ranjanmannige/backmap/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd backmap
uv pip install .
```
